Gnumeric 1.12.6 aka "TBD" is now available.

The Gnumeric Team is pleased to announce the availability of Gnumeric version 1.12.6. If you use a development version of gtk+ you will want to upgrade.

Noteworthy news in this release:

  • A redraw problem with gtk+ 3.9.x has been fixed.
  • A performance problem with large sheets has been fixed.

Details:

  • Andreas
    • Fix crash on corrupted files. [Bug 706526].
  • Morten
    • Fix crash on corrupted files. [Bug 706413] [Bug 706417].
    • Fix save style performance issue. [Bug 699045].
